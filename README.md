# SYNOPSIS

Sparrowdo module to install Nginx web server.

# Description

This module Makes a simple Nginx install ( default virtual host binded to 80 port ).

# Install

    $ panda install Sparrowdo::Nginx


# Usage

    $ cat sparrowfile

    module_run 'Nginx';

# Platforms supported

This is where I tested this module.

* CentOS
* Ubuntu
* Debian
